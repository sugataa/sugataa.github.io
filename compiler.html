Playing with Fire

Tue 17 December 2013

It's my birthday! But instead, I've decided to give you a present.

Compilers have always been a very mysterious thing to me, I've almost always viewed them as black magic. As Bill O'Reilly would say 'Source code goes in object code comes out. You can't explain that.'

Well, I set out to do exactly that, and reading multiple tutorials and installing various packages, I managed to create a simple toy language - it works exactly like a calculator. But the real magic is in how to take the arithmetic and output the results.
